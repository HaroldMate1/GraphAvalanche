# Documentation\n\n## Quick Start\n\nSee [README.md](../README.md) for installation and basic usage.\n\n## API Reference\n\n### CitationNetwork\n\nMain class for building and managing citation networks.\n\n#### Methods\n\n- `__init__(papers, batch_size=50)`: Initialize with paper list\n- `build_doi_index()`: Create DOI to paper ID mapping\n- `build_connections()`: Fetch and link citations\n- `build_graph()`: Create NetworkX directed graph\n- `get_stats()`: Return network statistics\n\n### MetadataLoader\n\nUtility class for loading and processing paper metadata.\n\n#### Static Methods\n\n- `load_excel(filepath)`: Load from Excel file\n- `load_csv(filepath)`: Load from CSV file\n- `validate_papers(papers)`: Check required fields\n- `clean_papers(papers)`: Standardize field names\n\n## Examples\n\nSee the `examples/` directory for sample code.\n\n## Performance Tips\n\n1. Use batch processing (default batch_size=50)\n2. Cache OpenAlex results locally\n3. Run on large datasets overnight\n4. Monitor API rate limits\n
